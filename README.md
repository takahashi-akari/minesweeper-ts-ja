## How to play
- Click on a cell to reveal it
- Right click on a cell to flag it
- If you reveal a mine, you lose
- If you flag all the mines, you win

## How to run
- Clone the repository
- Run `npm install`
- Run `npm start`
- Open `http://localhost:3000/` in your browser
